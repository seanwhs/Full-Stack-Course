# Full-Stack Course Roadmap (34 Days)

This roadmap outlines the complete **34-day Full-Stack Web Development Bootcamp**, from fundamentals to production-ready Wagtail sites. Each day links to detailed notes stored in this repository.

---

## Phase 1: Web Fundamentals

### Day 1 – Web Development Fundamentals
- Intro to web development concepts: HTML, CSS, JS overview, frontend/backend distinction  
- [Day 1 Notes](./1-Day1-Web-Dev-Fundamentals.md)

### Days 2–4 – HTML Deep Dive
- Day 2: HTML structure, semantic elements, forms, tables  
- Day 3: HTML accessibility, multimedia, meta tags  
- Day 4: HTML best practices, responsive foundations  
- [Day 2–4 Notes](./2-Day2-HTML-Deep_Dive.md)

### Days 5–7 – CSS Mastery
- Day 5: CSS fundamentals, box model, cascade, specificity  
- Day 6: Flexbox mastery, layout patterns, centering  
- Day 7: CSS Grid, responsive design, mobile-first workflow  
- [Day 5–7 Notes](./3-Day5-CSS-Mastery.md)

## Phase 2: Development Tools

### Days 8–9 – Git & Version Control
- Day 8: Git fundamentals, commits, branches, local workflow  
- Day 9: Collaboration, pull requests, merge conflicts, remote workflow  
- [Day 8–9 Notes](./4-Day8-Git-Bootcamp.md)

## Phase 3: Frontend Programming

### Days 10–14 – JavaScript Bootcamp
- Day 10: JS syntax, variables, operators, control flow, functions  
- Day 11: DOM manipulation, event handling, interactive elements  
- Day 12: Data handling, arrays, objects, JSON, fetch API  
- Day 13: Advanced JS concepts, scope, closures, ES6 features, modules  
- Day 14: Mini Project integrating HTML/CSS/JS  
- [Day 10–14 Notes](./5-Day10-JavaScript-Bootcamp.md)

### Days 15–19 – Python Bootcamp
- Day 15: Python fundamentals – variables, types, control flow, functions  
- Day 16: Data structures – lists, tuples, dicts, sets, comprehensions  
- Day 17: Functions & modules – scope, built-in functions, modular design  
- Day 18: File I/O & error handling – JSON, CSV, exceptions, logging  
- Day 19: Mini project integrating Python logic and data handling  
- [Day 15–19 Notes](./6-Day15-Python-Bootcamp.md)

### Days 20–24 – React Bootcamp
- Day 20: React fundamentals & JSX, functional components  
- Day 21: State & props, unidirectional data flow  
- Day 22: Event handling, controlled forms  
- Day 23: Lists, conditional rendering, API integration  
- Day 24: Mini project & component architecture  
- [Day 20–24 Notes](./7-Day20-React-Bootcamp.md)

---

## Phase 4: Backend Development

### Days 25–29 – Django Bootcamp
- Day 25: Django fundamentals, project setup, apps, dev server  
- Day 26: Models & database, ORM, migrations, relationships  
- Day 27: Views & templates, dynamic rendering, template tags  
- Day 28: Forms, validation, CRUD functionality  
- Day 29: Mini project integrating models, views, templates, CSS, JS  
- [Day 25–29 Notes](./8-Day25-Django-Bootcamp.md)

### Days 30–34 – Wagtail Bootcamp
- Day 30: Wagtail fundamentals & setup, admin interface  
- Day 31: Page models & content management, snippets  
- Day 32: Templates, StreamField, media handling  
- Day 33: Forms, search, workflows, permissions  
- Day 34: Mini project & production deployment  
- [Day 30–34 Notes](./9-Day30-Wagtail-Bootcamp.md)

---

> This roadmap is designed for sequential learning. Each phase builds upon the previous, preparing students for full-stack web development with professional workflows, production-ready applications, and CMS integration.
