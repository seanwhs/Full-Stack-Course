# ‚öõ 5-Day React Bootcamp

## From Components to Dynamic UIs

**Prerequisites:**

* HTML & CSS mastery
* JS fundamentals, DOM manipulation, fetch API
* Python backend logic for later API integration

**Learning Philosophy:**

* React = **component-driven UI**
* Emphasize **state, props, and unidirectional data flow**
* Build maintainable, modular frontend architecture
* Prepare for integration with Django REST API

**Format:** ~8 hours/day

* 30% lecture/concept
* 60% guided coding
* 10% reflection/code critique
